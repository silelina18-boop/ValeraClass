1. Тестирование функциональности попыток
Проверка ограничения попыток:

    Тест 1: Ответьте неправильно 2 раза на один вопрос 

        Ожидаемый результат: После 2-й ошибки кнопка становится красной с текстом "✗ Попытки исчерпаны"

    Тест 2: Попробуйте ответить в 3-й раз

        Ожидаемый результат: Появляется модальное окно "Попытки исчерпаны"

Проверка правильных ответов:

    Тест 3: Ответьте правильно с 1-й попытки

        Ожидаемый результат: Кнопка становится зеленой с текстом "✓ Отвечено верно"

    Тест 4: Ответьте правильно со 2-й попытки

        Ожидаемый результат: Система должна засчитать правильный ответ

3. Практическое тестирование в браузере
Быстрый тест всех правильных ответов:

    Откройте консоль браузера (F12)

    Вставьте этот код для автоматического выбора правильных ответов:

javascript

// Автоматически выбирает все правильные ответы
function autoSelectCorrectAnswers() {
    // Вопрос 1
    document.querySelector('input[value="bednye_lyudi"]').checked = true;
    // Вопрос 2
    document.querySelector('input[value="sevastopolskie rasskazy"]').checked = true;
    // Вопрос 3
    document.querySelector('input[value="beliy pudel"]').checked = true;
    // Вопрос 4
    document.querySelector('input[value="mozart1"]').checked = true;
    // Вопрос 5
    document.querySelector('input[value="alexandra_vladimirovna_vasnezova"]').checked = true;
    // Вопрос 6
    document.querySelector('input[value="russkiy_musey"]').checked = true;
    // Вопрос 7
    document.querySelector('input[value="zima"]').checked = true;
    // Вопрос 8
    document.querySelector('input[value="rimskie_kaniculi"]').checked = true;
    // Вопрос 9
    document.querySelector('input[value="liverpul"]').checked = true;
    // Вопрос 10
    document.querySelector('input[value="1872"]').checked = true;
    
    console.log("Все правильные ответы выбраны!");
}
autoSelectCorrectAnswers();

Тестирование пограничных случаев:

Тест без выбора ответа:

    Нажмите кнопку "Проверить ответ" не выбирая вариант

    Ожидаемый результат: Alert "Пожалуйста, выберите правильный вариант!"

Тест последовательности:

    Ответьте неправильно на вопрос 1

    Ответьте правильно на вопрос 2

    Вернитесь к вопросу 1 и попробуйте ответить еще раз

    Ожидаемый результат: Должна сохраняться информация о предыдущих попытках

4. Тестирование на разных устройствах
Проверьте:

    На компьютере: Работает ли с мышью и клавиатурой

    На телефоне: Корректно ли отображаются изображения и аудио

    В разных браузерах: Chrome, Firefox, Safari, Edge

Тест адаптивности:

    Измените размер окна браузера

    Проверьте, не ломается ли верстка

5. Тестирование аудио и изображений
Аудио-тесты:

    Нажмите play на всех аудиофайлах

    Проверьте громкость и качество звука

    Убедитесь, что можно ставить на паузу

Изображения:

    Проверьте загрузку всех картинок

    Убедитесь, что есть альтернативный текст (alt)

    Проверьте tooltips (title атрибуты)

    Откройте F12 → Console

    Проверьте наличие ошибок JavaScript

    Используйте console.log для отладки

// Проверить состояние всех вопросов
